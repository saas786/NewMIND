**Циклы** - повторяемые блоки кода 

```

while <условие>:	#пока условие True
	<блок кода>

else:				#иначе
	<блок кода>
	
for <переменная> in <список>:		# in range (start,stop,step)
	<блок кода>
	
	
for _ in range(): 		#используйте _ если переменная не нужна
	<блок кода>
```
